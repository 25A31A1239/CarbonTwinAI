# Carbon Twin AI 🌱

## Overview

Carbon Twin AI is a web application that helps users understand, track, and reduce their carbon footprint through personalized insights, eco challenges, and a dynamic Carbon Twin visualization.

## Features

* Carbon Footprint Tracking
* Carbon Twin Visualization
* Personalized Recommendations
* Future Impact Prediction
* Eco Challenges
* Statistics Dashboard
* Dark Mode Support

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Local Storage
